露脸云接口调试工具api文档

  1. 获取token，POST方式

API
/api/token
请求样例
app_info = {"app_id": "203228","app_secret": "fbe54068c5eafd5262ff169a1682cca2"}

requests.post('http://127.0.0.1:5000/api/token', data=app_info)
返回样例
{'Authorization': 'Bearer 5673fd6d-e841-48d1-9d26-4dc240214abd'}

  1. 创建用户，POST方式
API
/api/user/create
请求样例
data = {"token": "{'Authorization': 'Bearer 5673fd6d-e841-48d1-9d26-4dc240214abd'}", "app_user_id": "TEST_testAccount111"}

r = requests.post('http://127.0.0.1:5000/api/user/create', data=data)
返回样例
{'error_code': 2000, 'user': {'id': 'TEST_testAccount111', 'session_id': '746431ed0b3b4f3ab8d71447f2f0a430'}}

  1. 更新Session，POST方式
API
/api/user/session

data = {"token": "{'Authorization': 'Bearer 5673fd6d-e841-48d1-9d26-4dc240214abd'}", "app_user_id": "TEST_testAccount111"}
